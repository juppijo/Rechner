Lektion 01 / 06

Was ist HTML? 🏗️

HTML steht für HyperText Markup Language. Es ist die Grundstruktur jeder Webseite – wie das Skelett eines Gebäudes.

HTML verwendet sogenannte Tags (Etiketten), die dem Browser sagen, wie Inhalte dargestellt werden sollen.

HTML
<!-- Kommentar: Das ist ein einfaches HTML-Dokument -->
<!DOCTYPE html>
<html>
  <head>
    <title>Meine erste Seite</title>
  </head>
  <body>
    <h1>Hallo Welt!</h1>
    <p>Meine erste Webseite 🎉</p>
  </body>
</html>
💡 Tipp: Jedes Tag hat einen öffnenden Teil <tag> und einen schließenden Teil </tag>.

Überschriften & Text ✍️

HTML hat 6 Überschriftsebenen (h1 bis h6). h1 ist die größte, h6 die kleinste. Für normalen Text nutzt man <p> (Paragraph).

HTML
<h1>Größte Überschrift</h1>
<h2>Zweitgrößte Überschrift</h2>
<h3>Drittgrößte Überschrift</h3>

<p>Dies ist ein normaler Absatz.</p>
<p>Text kann <strong>fett</strong> oder <em>kursiv</em> sein.</p>

Lektion 03 / 06

Links & Bilder 🔗🖼️

Mit dem <a>-Tag erstellst du klickbare Links. Das href-Attribut gibt die Zieladresse an. Bilder werden mit <img> eingebunden.

HTML
<!-- Ein einfacher Link -->
<a href="https://google.de">Google öffnen</a>

<!-- Link in neuem Tab öffnen -->
<a href="https://google.de" target="_blank">
  Google (neuer Tab)
</a>

<!-- Ein Bild einfügen -->
<img src="bild.jpg" alt="Beschreibung" width="200">
💡 Tipp: Das alt-Attribut bei Bildern ist wichtig für die Barrierefreiheit – es beschreibt das Bild für Screenreader.

Lektion 04 / 06

CSS – Styling ✨

CSS (Cascading Style Sheets) macht HTML schön! Mit CSS kannst du Farben, Schriftarten, Abstände und viel mehr anpassen.

HTML + CSS
<!-- CSS direkt im HTML (style-Attribut) -->
<p style="color: blue; font-size: 20px;">
  Blauer Text
</p>

<!-- CSS im <style>-Block -->
<style>
  h2 {
    color: hotpink;
    background: yellow;
    padding: 10px;
    border-radius: 8px;
  }
</style>

Lektion 05 / 06

JavaScript – Interaktion ⚡

JavaScript macht Webseiten lebendig! Mit JS kannst du auf Klicks reagieren, Texte ändern, Animationen starten und vieles mehr.

HTML + JavaScript
<button onclick="sageHallo()">Klick mich!</button>
<p id="ausgabe">Hier erscheint der Text</p>

<script>
  function sageHallo() {
    // getElementById findet das Element
    let element = document.getElementById("ausgabe");
    element.textContent = "Hallo! Du hast geklickt! 🎉";
    element.style.color = "green";
  }
</script>
